## 📗 프로젝트 설명

```
⏰ 공공데이터를 이용한 미세먼지 알림이입니다. 
   셀렉박스를 통해 선택한 지역의 미세먼지 정보를 보여줍니다.
```
- 프로젝트는 모바일 사이즈로 만들어졌습니다.

## 📌 구현기능

- 공공데이터 api 로 데이터 받아오기
- 기본 지역 보기/ 전체 보기 / 즐겨찾기 페이지 구현
- 미세먼지 수치에 따른 카드 색상 변화
- 로컬스토로지를 이용한 즐겨찾기 기능 구현
